The Arbitrary Precision Calculator is designed to handle calculations involving very large numbers that cannot be stored using standard C data types like int or long long.

The project represents numbers using linked lists, where each node stores a single digit. Arithmetic operations are performed digit-by-digit similar to manual calculations.

✨ Features

➕ Addition of large integers

➖ Subtraction with borrow handling

✖ Multiplication using partial products

➗ Division using repeated subtraction

🔢 Supports numbers of unlimited size

➕➖ Handles positive and negative numbers

🧹 Removes leading zeros automatically

⚡ Efficient digit-wise processing using linked lists
